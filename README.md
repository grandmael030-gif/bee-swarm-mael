## 🚀 Déploiement en ligne (Optionnel)

Pour rendre le jeu accessible depuis n'importe où via une URL :

### Méthode 1: Netlify (Recommandé)

1. **Installez Node.js** depuis <https://nodejs.org>
2. **Ouvrez PowerShell/CMD** et exécutez :
   ```powershell
   npm install -g netlify-cli
   netlify login
   netlify deploy --prod --dir="c:\Users\Utilisateur\Documents\bee-swarm-simulator-cheater\bee-swarm-web"
   ```

3. **Récupérez l'URL** affichée (ex: `https://bee-swarm-grandmael.netlify.app`)

### Méthode 2: GitHub Pages (Gratuit)

1. Créez un compte sur <https://github.com>
2. Créez un nouveau repository
3. Uploadez les fichiers du dossier `bee-swarm-web`
4. Allez dans Settings → Pages → Source → Deploy from a branch → Main
5. Votre jeu sera disponible à `https://[votre-username].github.io/[nom-repo]/`
