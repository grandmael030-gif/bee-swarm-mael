# 🐻 Les 5 Bears de Bee Swarm Simulator Web

## 1. BRUNO LE BOURDON 🐝
**Titre :** L'Apprenti Butineur  
**Localisation :** La Prairie Dorée (centre de la map)

### Apparence
- Couleur : Brun (#8B4513)
- Taille : Moyenne
- Accessoires : Chapeau de paille trop grand, panier de fleurs
- Description : Ours maladroit mais gentil, toujours prêt à aider

### Personnalité
Gentil, lent, encourageant. Parle avec des "Euh..." et des "Hmm..."

### Déblocage
Disponible dès le début du jeu

### Quêtes (5)

**Quête 1 : Premiers Pas**
- Objectif : Collecter 50 pollen
- Dialogue : "Euh... commence par collecter un peu de pollen. 50 unités devraient suffire !"
- Récompense : 100 miel, 10 XP

**Quête 2 : Première Abeille**
- Objectif : Convertir 200 pollen en miel
- Dialogue : "Hmm... avec du miel, tu peux acheter des upgrades. Fais 200 miels !"
- Récompense : 150 miel, 15 XP

**Quête 3 : Collectionneur**
- Objectif : Acheter 1 upgrade Capacity
- Dialogue : "Euh... il te faut plus de place ! Achète l'upgrade Capacity une fois."
- Récompense : 250 miel, 20 XP

**Quête 4 : Explorateur**
- Objectif : Visiter 10 fleurs différentes
- Dialogue : "Hmm... va voir les fleurs ! Collecte du pollen de 10 fleurs différentes."
- Récompense : 300 miel, 25 XP + 1 abeille Common

**Quête 5 : Le Vrai Butineur**
- Objectif : Collecter 500 pollen d'un coup
- Dialogue : "Euh... dernière épreuve : collecte 500 pollen d'un coup !"
- Récompense : 500 miel, 50 XP + 1 abeille Rare

```
BRUNO (Début) → LUNA (Après Bruno Q2) → RUSTY (Après Luna) → MISTY (Après Rusty) → EMBER (Après tous)
```

## 💰 Récompenses Totales

Si un joueur complète TOUTES les quêtes :
- **Miel total** : 36 150
- **XP total** : 1 195
- **Abeilles** : 2 Common, 3 Rare, 3 Epic, 5 Legendary
- **Titre final** : Maître des Abeilles

## 🔧 Intégration Technique

Le fichier `bears_system.js` contient :
- `BearSystem` : Classe pour gérer les quêtes
- `BEARS_DATA` : JSON avec tous les bears et quêtes
- Sauvegarde automatique dans localStorage
- Méthodes pour vérifier la progression, donner les récompenses

Pour l'intégrer dans `game.js` :
```javascript
// Dans le constructeur Game
this.bearSystem = new BearSystem();

// Quand le joueur collecte du pollen
this.bearSystem.updateProgress('collect_pollen', amount);

// Pour afficher les bears disponibles
const unlockedBears = this.bearSystem.getUnlockedBears();
```
